1. Rotate

Spin element around its center.

Rotate
transform: rotate(180deg);

2. Pulse

Scale element up and down continuously.

Pulse
@keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.3); } 100% { transform: scale(1); } }

3. Bounce

Element bounces up and down.

Bounce
@keyframes bounce { 0%, 100% { translateY(0); } 25% { translateY(-30px); } 75% { translateY(-15px); } }

4. Slide In

Element slides in from the left.

Slide
@keyframes slideIn { from { transform: translateX(-100%); } to { transform: translateX(0); } }

5. Skew

Distort element along axes.

Skew
@keyframes skewAnimation { 0% { transform: skewX(0deg) skewY(0deg); } 50% { transform: skewX(20deg) skewY(10deg); } 100% { transform: skewX(0deg) skewY(0deg); } }

6. Combined

Multiple transforms together.

Multi
transform: rotate(90deg) scale(1.2) translateY(-10px);

7. 3D Rotation

Rotate in 3D space.

3D
transform: rotateX(45deg) rotateY(45deg); transform-style: preserve-3d;
